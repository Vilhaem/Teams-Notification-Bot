# Teams-Notification-Bot
<a href = "https://pstn-test.azurewebsites.net/swagger/index.html" target="_blank">This link</a> is just for demoing the swagger UI used for debugging
### <b>Notification Bot that calls user via teams and plays a text-to-speech clip</b>

Can be extened for Public Switched Telephone Network(PSTN) calls and Dual-Tone Multi-Frequency(DTMF).
<br>Requires Azure AD tenant, M365 licence(Teams) in order to function properly. 
<br>PSTN functionality will require a <a href="https://learn.microsoft.com/en-us/graph/cloud-communications-phone-number#assign-a-phone-number-to-your-bot" target="_blank">Azure AD application registered with a phone number</a>.
